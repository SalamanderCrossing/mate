You can use this to: - Get the current CLI command - Save the run to a JSON file - Get the current save directory - Get the current checkpoint path

Example

This is not meant to replace a proper logging framework, but rather to provide a simple way to save the results of an experiment. This is especially useful when you want to compare multiple experiments (see mate.results()).

If you want, with pytorch lightning, you can directly pass the logged_metrics dictionary to this function.
